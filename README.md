# 📐 AutoCAD Design Project

A CAD design project containing the complete source files and documentation.

---

## 📂 Project Structure

```
.
├── our project.FCStd
├── our projectnew.FCStd
├── Images/
│   └── Preview.png
└── README.md
```

---

## 📁 Project Files

| File | Description |
|------|-------------|
| `our project.FCStd` | Original CAD design file |
| `our projectnew.FCStd` | Updated/revised version of the design |

---

## 🛠 Software Used

- FreeCAD (.FCStd format)
- Exportable to STEP, STL, DXF, and other CAD formats

---

## 🚀 Opening the Project

1. Install **FreeCAD**.
2. Open either:
   - `our project.FCStd`
   - `our projectnew.FCStd`
3. View or modify the design as needed.
